# E-Commerce Web Application

A modern **E-Commerce web application** built with **React.js** that allows users to browse products, add items to the cart, and simulate an online shopping experience.

## Features

- Browse a catalog of products with images, descriptions, and prices.
- Add products to cart and manage cart items (update quantity, remove items).
- Search and filter products.
- Responsive design for desktop and mobile devices.
- Easy to extend with payment gateway integration in the future.

## Technologies Used

- **Frontend:** React.js, HTML5, CSS3, JavaScript (ES6+)
- **State Management:** React Context API / Redux (if used)
- **Routing:** React Router
- **Styling:** CSS Modules / Styled Components / Tailwind CSS (choose what you used)
- **Optional:** Local storage for cart persistence
